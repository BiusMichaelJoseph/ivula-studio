

# Ivula Technologies — "Tech Studio" Website Rebuild

## Brand & Visual Direction
- **Dark-mode-first** aesthetic with a **Deep Violet** accent palette
- Clean, minimalist, premium typography with generous whitespace
- Core message: *"We build scalable software 5x faster."*
- Glassmorphism effects on navigation and key UI elements

---

## Pages & Sections

### 1. Home Page
- **Floating glassmorphism navbar** — sticky, blurred background, Ivula logo + nav links (Home, Portfolio, Process, Pricing, Contact)
- **Hero section** — bold headline, animated "MVP Delivery" timeline showing days-to-launch, CTA button ("Let's Build"), subtle gradient glow effects
- **Services grid** — 4 cards (Web Development, Mobile Apps, Automation, AI Integration) with icons and short descriptions
- **Client logo ticker** — scrolling row of partner/client logos
- **Stats bar** — key metrics like "50+ Projects," "5x Faster Delivery," "98% Client Retention"

### 2. Portfolio / Case Studies
- **Filterable gallery** with category tags (Web, Mobile, AI, SaaS)
- **4–6 project cards** including iSpaces and iB360, each showing:
  - Project thumbnail/mockup
  - "Idea to Launch" metric (e.g., "Launched in 3 weeks")
  - Tech stack badges
  - Brief outcome description
- Click-through to expanded case study detail view

### 3. The Ivula Process
- **Horizontal scroll section** with 4 stages:
  1. **Discovery** — Understanding your vision
  2. **Prototyping** — Rapid wireframes & design
  3. **Development** — Agile sprints, weekly demos
  4. **Scale** — Launch, optimize, grow
- Each stage has an icon, description, and estimated timeline
- Smooth scroll-triggered animations as user progresses

### 4. Pricing / Packages
- **3-tier pricing cards**: Starter, Startup MVP Bundle (highlighted/featured), Enterprise
- The **Startup MVP Bundle** is visually emphasized with a "Most Popular" badge and violet glow
- Each card lists deliverables, timeline, and a CTA
- FAQ accordion below for common questions

### 5. Contact / Lead Gen
- **Multi-step form** (3 steps):
  1. "What are you building?" — project type selection
  2. "Estimated Budget?" — range selector
  3. "How fast do you need to launch?" — timeline + name/email/phone
- Progress indicator at top, smooth step transitions
- Success screen with confirmation message (frontend-only, no database)

### 6. Footer
- Company info, quick links, social media icons, WhatsApp/Telegram links
- "© Ivula Technologies" branding

---

## Cross-Cutting Features
- **Mobile-first responsive design** — feels like a native app on phones
- **Smooth scroll animations** — fade-in, scale, slide effects on all sections triggered by scroll position
- **Dark mode by default** with Deep Violet accents throughout
- **Single-page app** with smooth scrolling between sections (no page reloads)
- All built with React, Tailwind CSS, and shadcn/ui components
